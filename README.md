# Fluxi
Fluxi is a Personal Kanban Board that helps manage the workflow of your everyday tasks.

## Programs
* Backend:
    * Python
    * Flask

* Frontend:
    * HTML5
    * CSS3
    * Javascript
    * Bootstrap 4
    * jQuery/AJAX

* Database:
    * SQLite


## Fuctionality
* Users
    * Register user
    * Login user
    * Logout user
    * View user page
* Boards
    * Create board
    * View board
    * Edit board name
    * Delete board
* Cards
    * Create card
    * Move card between lists
    * View card
    * Edit card text
    * Edit card description
    * Edit card due date
    * Delete card
* Notes
    * Create note in card
    * Edit note text
    * Delete note
